# @ztrehagem/phrase

It generates ramdom string with `[A-Za-z0-9]`

## Use as CLI

```
$ npm i -g @ztrehagem/phrase
$ phrase
aQb4FWYZ4tv9uZID
$ phrase 8
zYZ4XwQz
```

## Use as Module

```js
const phrase = require('@ztrehagem/phrase')

phrase(16) //=> 'cMvUEYGzKVDqDhSL'
phrase(8) //=> 'KLEoBFA9'
```
